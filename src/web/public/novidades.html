Versão 3.1

Novidades e Mudanças
  • Mudanças internas no serviço de banco de dados.

Versão 3.0

Novidades e Mudanças
  • NOVO! Use mecanismos de busca (como o Google) quando o bot não achar uma palavra. Essa função será mais aprimorada e ficará mais estável ainda na próxima atualização.
  • NOVO! Ao usar o modo inline, as palavras recebem um ícone com a primeira letra.
  • NOVO! O site de apresentação do bot foi completamente refeito.
  • NOVO! Ajude o desenvolvimento do bot pelo comando /donate.
  • Não há mais necessidade de enviar / quando se quiser usar um atalho no privado. Ainda funciona, mas é possível mandar apenas a palavra e receber definições, sinônimos ou exemplos. Lembrando que você pode escolher um dos três para ser seu atalho nos ajustes.
  • O comando de ajuda foi melhorado e inclui um FAQ.
  • O comando de ajustes foi reorganizado e inclui a opção de adicionar ou remover mecanismos de busca.
Correções de Bugs
  • O código do bot foi completamente refeito.
  • O atalho para exemplos não estava funcionando na versão anterior (2.1).
  • Cancele qualquer mensagem do tipo “Respondendo essa mensagem, […]” usando o comando /cancelar.
  • Todas as frases do bot foram revistas e melhoradas.

Versão 2.1

Novidades e Mudanças
  • NOVO! Comando /exemplos. Veja frases exemplificando o uso de uma palavra.
  • Escolha os exemplos como seu atalho padrão no chat privado com o bot.
  • Gerencie os dados que o bot tem sobre você no comando de configurações (/settings). O bot guarda o seu ID do Telegram para recebimento de novidades e as suas configurações, saiba mais.
Correções de Bugs
  • Palavras acentuadas agora podem ser encontradas no dicionário.
  • Palavras com grafia igual, mas acentuação diferente estavam recebendo os mesmos significados (como, por exemplo, metro e metrô).
  • Clique do botão no modo inline ao não ter respostas agora funciona.
  • Alguns textos e instruções foram modificados para se padronizarem.

Versão 2.0

Novidades e Mudanças
  • NOVO! Agora é possível apenas enviar /definir ou /sinonimo e escrever uma palavra respondendo a mensagem do bot.
  • NOVO! Configure atalhos para definição ou sinônimos de uma palavra no chat privado. Por exemplo, você pode digitar /arroz para receber a definição de arroz.
  • NOVO! Comandos auxiliares como /settings e /help foram adicionados.
  • A ordem dos comandos e a descrição de todos foram atualizadas.
  • Ao invés de obter apenas um sinônimo, obtenha uma lista com todos.
  • A política de privacidade foi atualizada.
Correções de Bugs
  • Os sinônimos estão de volta no modo inline.
